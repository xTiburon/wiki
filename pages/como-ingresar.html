🔌 Cómo Ingresar

Requisitos previos

Para conectarte a PlanetMC necesitas tener Minecraft Java Edition 1.20+ o Bedrock Edition actualizado. Ambas versiones son compatibles y pueden jugar juntas en el mismo servidor.

Java Edition

Sigue estos pasos para conectarte desde Java:

  1. Abre el launcher de Minecraft y selecciona la versión 1.20.x o superior.
  2. En el menú principal, haz clic en Multijugador.
  3. Haz clic en Añadir servidor.
  4. En el campo "Dirección del servidor" escribe: play.planetmc.net
  5. Guarda y haz doble clic en el servidor para conectarte.
Recomendamos usar Fabric + Sodium para mayor rendimiento. Consulta la lista de mods permitidos.

Bedrock Edition

Para conectarte desde Bedrock (móvil, consola o Windows 10/11):

  1. Abre Minecraft y ve a Jugar → Servidores → Añadir servidor.
  2. Ingresa los siguientes datos:
DIRECCIÓN bedrock.planetmc.net Haz clic para copiar
PUERTO 19132 Puerto estándar de Bedrock
En consolas como PS4/PS5 o Nintendo Switch es posible que necesites un método adicional para conectarte a servidores externos. Consulta los tutoriales específicos en nuestro Discord.

¿Problemas para conectarte?

Si no puedes ingresar, verifica:

  • Que tienes una versión compatible (1.20 o superior para Java).
  • Que la IP está escrita correctamente, sin espacios extras.
  • Que el servidor no está en mantenimiento (revisa nuestro Discord).

Si el problema persiste, abre un ticket en nuestro servidor de Discord.
