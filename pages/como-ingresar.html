Cómo Ingresar a PlanetMC

PlanetMC es compatible tanto con Java Edition como con Bedrock Edition. Selecciona tu plataforma y sigue los pasos.

🎮 Elige tu plataforma

☕
Java Edition
Para PC con el launcher oficial, TLauncher o cualquier launcher Java. Incluye Premium y No Premium.
📱
Bedrock Edition
Para móvil (Android / iOS), Windows 10/11, Xbox, PlayStation y Nintendo Switch.

🌐 Datos de conexión rápida

Java IPplay.planetmc.net 📋
Puerto Java25565 (predeterminado)
Versión1.21.x
Bedrock IPbedrock.planetmc.net 📋
Puerto Bedrock19132 📋
Versión1.21.x
⚠️
Versión exacta: Asegúrate de usar la versión 1.21.x. Una versión diferente puede impedir la conexión.
